# fund

基金股票查询插件。支持查询场外基金、场内 ETF/LOF、A股个股和指数的实时行情和历史数据。

## 功能特性

- **场外基金**：查询基金净值、阶段收益、十大重仓股
- **场内 ETF**：查询实时价格、涨跌幅、历史走势
- **场内 LOF**：查询实时价格、涨跌幅、历史走势
- **A股个股**：查询股票实时行情（需带交易所后缀）
- **指数**：查询上证指数、深证成指等主要指数
- 智能缓存机制，减少 API 请求
- 支持多数据源自动切换（东方财富 → 同花顺）
- 支持分群开关控制

## 代码格式说明

| 类型 | 格式 | 示例 |
|------|------|------|
| 场外基金 | 6位数字（00-09开头） | `018957`、`002170` |
| 场内 ETF | 6位数字（51x/58x/56x/55x/15x） | `510300`、`159915` |
| 场内 LOF | 6位数字（50x/16x） | `163406`、`501018` |
| A股（沪市） | 6位数字.SH | `600000.SH` |
| A股（深市） | 6位数字.SZ | `000001.SZ` |
| A股（北交所） | 6位数字.BJ | `920029.BJ` |
| 指数（上证） | 6位数字.SH | `000001.SH` |
| 指数（深证） | 6位数字.SZ | `399001.SZ` |

## 配置项

在 `.env` 或 `.env.prod` 中配置：

### 基础配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `fund_plugin_enabled` | bool | true | 是否启用基金查询插件 |
| `fund_group_mode` | str | "all" | 群组控制模式：all/whitelist/blacklist |
| `fund_group_whitelist` | list | [] | 白名单群组 |
| `fund_group_blacklist` | list | [] | 黑名单群组 |

### 数据获取配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `fund_history_days` | int | 30 | 获取历史数据天数（1-365） |
| `fund_display_recent_days` | int | 7 | 显示最近天数（1-30） |

### 缓存配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `fund_cache_ttl_minutes` | int | 5 | 缓存有效期（分钟） |
| `fund_max_cache_size` | int | 100 | 最大缓存条目数 |

### 功能开关

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `fund_enable_etf` | bool | true | 是否启用 ETF 查询 |
| `fund_enable_lof` | bool | true | 是否启用 LOF 查询 |
| `fund_enable_stocks` | bool | true | 是否启用股票查询 |
| `fund_enable_off_market` | bool | true | 是否启用场外基金查询 |
| `fund_enable_index` | bool | true | 是否启用指数查询 |
| `fund_enable_data_source_fallback` | bool | true | 是否启用数据源切换 |
