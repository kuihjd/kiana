# un_nickname

群成员昵称与集合管理插件。支持为群成员添加昵称，或将多个成员归入一个集合，通过 `at昵称` 或 `at集合名` 快速 @ 对应的人。

## 功能特性

- 为群成员添加多个昵称，发送 `at昵称` 自动转换为 @
- 创建成员集合，发送 `at集合名` 一次性 @ 所有成员
- @ 集合时自动跳过发送者本人
- 昵称和集合名共享命名空间，先到先得，不允许重名
- 成员退群时自动清理其昵称和集合记录
- 集合为空时自动删除

## 命令列表

### 昵称管理

| 命令 | 说明 |
|------|------|
| `@某人 昵称 xxx` | 为某人添加昵称 |
| `@某人 昵称` | 查看某人的所有昵称 |
| `at昵称` | 自动转换为 @ 对应的人 |
| `删除昵称 @某人 xxx` | 删除某人的指定昵称 |
| `清空昵称 @某人` | 清空某人的所有昵称 |

### 集合管理

| 命令 | 说明 |
|------|------|
| `集合 xxx @人1 @人2` | 创建集合或向集合添加成员 |
| `集合 xxx` | 查看集合成员 |
| `集合列表` | 列出本群所有集合 |
| `at集合名` | 自动 @ 集合内所有成员（跳过发送者） |
| `移除集合 xxx @人` | 从集合中移除成员 |
| `删除集合 xxx` | 删除整个集合 |

## 使用示例

### 昵称功能

```
用户: @张三 昵称 老张
Bot: 昵称'老张'成功绑定到用户!

用户: at老张 吃饭了吗
Bot: @张三 吃饭了吗

用户: @张三 昵称
Bot: 该用户的昵称: 老张

用户: 删除昵称 @张三 老张
Bot: 成功删除昵称：老张
```

### 集合功能

```
用户: 集合 养猫人群 @张三 @李四 @王五
Bot: 已创建集合「养猫人群」，添加了 3 人: 张三、李四、王五

用户: 集合 养猫人群
Bot: 集合「养猫人群」共 3 人: 张三、李四、王五

用户: 集合列表
Bot: 本群共 1 个集合:
       • 养猫人群 (3人)

用户(张三): at养猫人群 周末一起撸猫
Bot: @李四 @王五 周末一起撸猫

用户: 移除集合 养猫人群 @李四
Bot: 已从集合「养猫人群」移除: 李四

用户: 删除集合 养猫人群
Bot: 已删除集合「养猫人群」（原有 2 人）
```

### 命名冲突

```
用户: 集合 老张 @某人
Bot: 名称「老张」已被昵称占用!

用户: @某人 昵称 养猫人群
Bot: 名称「养猫人群」已被集合占用!
```

## 配置项

在 `.env` 或 `.env.prod` 中配置：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `max_nickname_length` | int | 15 | 昵称最大长度 |
| `max_collection_name_length` | int | 15 | 集合名最大长度 |
| `max_collection_members` | int | 50 | 单个集合最大成员数 |

## 注意事项

- 昵称和集合名只能包含汉字、字母和数字
- 成员退群时，其昵称和集合记录会自动清理
- 当集合最后一个成员被移除或退群时，集合会自动删除
