# anime_trace

以图搜番插件。通过发送图片或回复包含图片的消息，自动识别图片中的动漫信息，返回番剧名称、集数、时间点和置信度。

## 功能特性

- 支持直接发送图片搜索动漫来源
- 支持回复包含图片的消息进行搜索
- 自动识别简体中文番名
- 返回匹配的集数和时间点
- 显示识别置信度
- 成人内容自动过滤视频预览

## 命令列表

| 命令 | 说明 | 权限 |
|------|------|------|
| `搜番 [图片]` | 发送图片搜索动漫来源 | 所有人 |
| `以图搜番 [图片]` | 同上 | 所有人 |
| `搜番`（回复消息） | 回复包含图片的消息进行搜索 | 所有人 |

## 使用示例

### 直接发送图片

```
用户: 搜番 [图片]
Bot: 识别结果：
     番名：进击的巨人
     第 1 集 12 分 34 秒
     置信度：87.65%
```

### 回复消息搜索

```
用户A: [发送一张动漫截图]

用户B: (回复用户A的消息) 搜番
Bot: 识别结果：
     番名：鬼灭之刃
     第 19 集 18 分 45 秒
     置信度：92.31%
```
