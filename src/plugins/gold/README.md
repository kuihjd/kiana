# gold

实时黄金价格查询插件。支持查询当前金价和查看金价走势图，数据来源于招商银行。

## 功能特性

- 实时查询黄金价格
- 支持自定义时间范围的走势图
- 定时记录金价数据，持久化存储
- 群聊冷却机制，防止刷屏
- 私聊无冷却限制
- 支持分群开关控制

## 命令列表

| 命令 | 说明 | 权限 |
|------|------|------|
| `金价` | 查询当前黄金价格 | 所有人 |
| `金价走势` | 查看默认时间范围的金价走势图 | 所有人 |
| `金价走势 [时间]` | 查看指定时间范围的走势图 | 所有人 |

### 时间格式支持

- 分钟：`30分钟`、`30分`、`30min`、`30m`
- 小时：`24小时`、`24时`、`24h`
- 天：`7天`、`7日`、`7d`
- 周：`1周`、`1星期`、`1w`
- 月：`1月`

## 使用示例

```
用户: 金价
Bot: 628.50

用户: 金价走势 24小时
Bot: [24小时金价走势图]

用户: 金价走势 7天
Bot: [7天金价走势图]

用户: 金价走势
Bot: [默认时间范围走势图]
```

## 配置项

在 `.env` 或 `.env.prod` 中配置：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `gold_plugin_enabled` | bool | true | 是否启用金价查询插件 |
| `gold_enable_price_query` | bool | true | 是否启用金价查询功能 |
| `gold_enable_chart` | bool | true | 是否启用金价走势图功能 |
| `gold_group_mode` | str | "all" | 群组控制模式：all/whitelist/blacklist |
| `gold_group_whitelist` | list | [] | 白名单群组 |
| `gold_group_blacklist` | list | [] | 黑名单群组 |
| `cooldown_time` | int | 1 | 冷却时间（秒） |
| `price_fetch_interval` | int | 600 | 金价获取间隔时间（秒） |
| `chart_window_hours` | int | 120 | 趋势图默认展示时间窗口（小时） |
| `price_history_limit` | int | 86400 | 内存中保留的历史数据最大数量 |
| `min_window_seconds` | int | 3600 | 趋势图最小时间窗口（秒） |
